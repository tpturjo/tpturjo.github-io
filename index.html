Project Name: Performance Review Web Application

Our four-member team, consisting of myself, Daniel, Caleb, and Tanjeet, is currently working on a project called the "Performance Review Web Application" as part of our COMP2005 course in Fall 2023. This web-based application is being developed using a stack of technologies including Python, HTML, CSS, Bottle (a Python web framework), and SQL for database management.

The goal of this project is to create a user-friendly tool that supports the process of conducting performance reviews within a team and maintains the data and information related to these reviews. As we are in the development phase, our vision includes setting up review schedules, checking the completion status of scheduled reviews, and conducting reviews on individual and team performance. Registered users will have access to their own reviews and the ability to enter new ones. Reviews may be structured with headings and checklists, and users can start draft reviews and return to them later before submitting a final version. Additionally, reviews could be published to other registered users, either with the author's identity or anonymously.

This project is a collaborative effort, and our four-member team is dedicated to delivering a practical and efficient tool for performance reviews. We are on track to complete it by the end of this semester and are excited to see it evolve with more features and enhancements as the project progresses.
